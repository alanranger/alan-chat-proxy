📊 Step-by-Step Guide to Using This Dashboard

🚀 First Time Setup (Do This Once):

  1. Go to Admin Panel: Click Admin Panel link below
  2. Run Analytics Aggregation: Click "Run Analytics Aggregation" button
  3. Wait for Success: You should see "✅ Analytics aggregation completed successfully"
  4. Return Here: Come back to this Analytics Dashboard

📈 Daily Usage (What Each Tab Shows):

  1. Overview Tab: See total sessions, questions, confidence scores, and response times
  2. Questions Tab: View most frequently asked questions and their confidence levels
  3. Sessions Tab: Browse individual chat sessions - search and click "View" for details
  4. Performance Tab: Monitor confidence and response time trends over time
  5. Insights Tab: Get automated recommendations for content improvements

🔄 When Data Looks Empty or Old:

  1. Check Admin Panel: Go to Admin Panel
  2. Run Manual Aggregation: Click "Run Analytics Aggregation" again
  3. Refresh This Page: Come back and refresh the dashboard

💡 Pro Tip: The system runs automatically every day at 1 AM, but you can trigger it manually anytime!
